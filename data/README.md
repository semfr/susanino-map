# data/ — единственный источник данных карты (single-source)

Этот каталог — **канон**. Его читают:

- **бот** (`bot/`, через `DATA_DIR`, по умолчанию `../data`);
- **сайт** — опосредованно: перед `dev`/`build` скрипт `web/scripts/sync-data.mjs`
  зеркалит этот каталог в `web/src/data/` (то зеркало в git игнорируется и правится автоматически).

**Правьте только файлы здесь.** Будущая админка через бота (вариант A: JSON + git)
тоже редактирует эти файлы → коммит → автo-пересборка сайта (GitHub Actions) + `git pull` на боте.

Файлы: `objects.json`, `categories.json`, `config.json`, `illustrations/`.
